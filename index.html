What is Business Intelligence?

A Decision Aiding Tool

Transforms data into something useful that can be analysed.

Front-end, Relies on data.

Business Intelligence Types

  1. Operational Reporting - Structured format. Requires understanding of user.
  2. Ad-hoc Reporting - Aid report creation.
  3. OLAP Analysis - Precalculated and prestructured information. Data navigation.
  4. Data Visualisation - Microdecisions, recently popular. Plug and play. Query intensive.
  5. Dashboards - High-level view allowing access to details. Compositions of previous methods.

Data Warehousing

What is Data Warehousing?

Back-end, behind-the-scenes.

Shared, decision making asset.

Develop, Reports & Analytics

Reading data. Data to information.

Data History - historical snapshots of when data changes. Allows the world to change and be tracked, unlike transaction system for example.

Data Scope - allows integrated view of system.

Data Structure - optimised for reading large amounts of data on demand.

Application Databases designed for data entry, usually one at a time, to be used by others. Data retrival of possibly lots of rows may slow it down for other users.

Avoids 'siloing' data.

Data Warehouse Qualities

Data in one location to simplify access.

Simplification and centralisation of query logic.
